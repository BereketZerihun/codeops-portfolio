Five terminal commands:
 1,mkdir - To create new folder
 2,touch - To create new file
 3,cd - To change directory
 4,clear - To clear the terminal
 5,mv - To move folder and file to another directory or to rename them

Three git commands:
 1,git init - To initialise a new local repo
 2,git status - To to see the current state of the working directory and also the staging area
 3,git add - To move changes from working directory to staging area
